# Simple Clasificador de Imagenes de perros o gatos (cambiable si es que se quiere clasificar 2 tipos de animales distintos)
## 1. Primero entrenar el modelo con imagenes creando una carpeta /imagenes/train aqui ingresar dos carpetas perros/gatos (ya precargadas) y una carpeta imagenes/test para guardar imagenes para testear el modelo
## 2. Comentar la parte de "#PREDICCION" y ejecutar el archivo
## 3. Al terminar todas las epoch y finalizar el entrenamiento comentar la parte de "#ENTRENAMIENTO" y cambiar el nombre de la ruta al nombre de la imagen que se quiera testear y ejecutar el archivo
